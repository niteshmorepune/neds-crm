The detail that sells itself

Built for how India actually invoices.

Most generic CRMs treat GST as an afterthought. This one was designed around it from day one — because we had to get it right for our own invoices first.

Automatic, correct, every time

  • Same state (Maharashtra): auto-splits CGST + SGST, half each
  • Different state: auto-applies IGST at full rate
  • Overseas clients: zero-rated "Export of Services" invoice, no tax
  • GSTIN validated against the official 15-character format before it's saved
  • Editable tax rate per line item (default 18%, not locked)

What the invoice looks like

  • Both parties' GSTIN shown, HSN/SAC code per line
  • Full tax breakup table + amount in words
  • Legal numbering: NEDS/2026-27/0042 — financial year + sequence
  • Bank details, UPI ID and a branded PDF, emailed automatically
  • Draft invoices stay invisible to the client until accounts hits "Send"

Cost of ownership

No servers to hire someone to babysit.

A deliberate technical choice, and a genuine selling point for price-sensitive Indian SMB prospects.

What it runs on

  • Standard shared web hosting (~₹5,000–15,000/year class of hosting) — no dedicated server, Docker, or DevOps hire required
  • No Redis, no message queue infrastructure, no websocket server to maintain
  • Standard MySQL database — any hosting provider, any competent developer can support it later

Why prospects should care

  • Dramatically lower running cost than a custom app built on "modern" infra (Kubernetes, managed queues, etc.)
  • No lock-in to us for hosting — it's plain PHP/MySQL, portable to any host
  • Predictable monthly cost — no surprise cloud bills that scale with traffic
